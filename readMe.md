## 工作问题

我是前面集中弄明白要需求、老的流程、新流程的实现方案，然后再做可维护性的修改，比如枚举、画流程图、idea书签，我觉得已经很高效了，但在第一次进项目的时候还是需要很长时间的适应，我觉得也归咎于我的项目能力少了

比如财务、税务尽调入库的字段控制，这里做的就很差，我从代码上都看不出这些字段有哪些，还要看入参才能知道哪些字段入库了哪些没有
